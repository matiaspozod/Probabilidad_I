# Probabilidad I

- **Profesor:** Johan Van Der Molen Moris

## Descripción del curso

En este primer curso de una secuencia de dos, el estudiantado conoce elementos de teoria de probabilidad que son fundamentales para el estudio y desarrollo del analisis estadistico. Entre ellos, estudian distribuciones de probabilidad en una dimension, sus principales propiedades, transformaciones y valores esperados. Las principales metodologias utilizadas son el aprendizaje basado en problemas y laboratorios computacionales, y las principales evaluaciones son controles y examen.

## Resultados de aprendizaje

1. Analizar el comportamiento de modelos estadisticos simples desde el punto de vista de la teoria de probabilidad para su aplicacion. 

2. Aplicar modelos probabilisticos a la resolucion de diversos problemas cotidianos. 

3. Simular fenomenos aleatorios para la comprension de su naturaleza, usando software estadistico.


## Contenidos

1. Modelo de Probabilidad y Variable Aleatoria.
    1. Conceptos y propiedades basicas. 
    2. Probabilidad condicional e independencia. 
    3. Teorema de Bayes. 
    4. Definicion y tipo de Variables Aleatoria. 
    5. Funcion de distribucion, funcion de probabilidad y funcion de densidad. 
    6. Simulacion de variables aleatorias y aplicaciones. 
 
2. Transformaciones y Valor Esperado. 
    1. Distribuciones de funciones de variables aleatorias. 
    2. Valores Esperados. 
    3. Momentos, funciones generadoras y funcion caracteristica. 
    4. Familias de distribuciones más utilizadas.
